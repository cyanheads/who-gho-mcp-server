# 0.1.10 — 2026-05-30

## Changed

- **`who_search_indicators`** — `totalMatches` and `note` removed from `output`; equivalent data now surfaces via the `enrichment` block as `effectiveQuery`, `totalCount`, and `notice`. Reaches both `structuredContent` and `content[]` without a `format()` entry.
- **`who_list_indicators`** — `total` and `hasMore` removed from `output`; now surfaces as `totalCount`, `hasMore`, and `pageInfo` in the `enrichment` block.
- **`who_query_indicator_data`** — `totalRows`, `truncated`, and `truncatedNote` removed from `output`; now surfaces as `totalRows`, `appliedFilters`, and `notice` in the `enrichment` block. A custom `enrichmentTrailer` renders the filter echo as formatted markdown in `content[]`.
- **`@cyanheads/mcp-ts-core`** ^0.9.13 → ^0.9.16 — picks up the `ctx.enrich` API, enrichment contract support, and `enrichmentTrailer` rendering.
- **Skills synced from framework** — `add-tool`, `add-service`, `api-context`, `api-linter`, `git-wrapup`, `maintenance`, `setup`, `polish-docs-meta`, `add-app-tool`, `design-mcp-server`.
